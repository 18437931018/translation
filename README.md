# 这是什么
     
   这是一个vscode的翻译插件，选中vscode内的文档右键点击翻译，即可在vscode内部侧边栏打开一个内嵌百度翻译的webview将刚才选中的文本翻译信息显示出来

# 怎么用
   * 将此项目down到本地的vscode插件目录，目录地址如下：
    ![path](https://github.com/18437931018/translation/blob/master/path.png)
   * 安装依赖 npm install
